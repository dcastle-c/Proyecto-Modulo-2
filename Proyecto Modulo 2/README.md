# Alke Wallet - The Castle Wallet

The Castle Wallet es una aplicación Front-end dinámica diseñada para una empresa Fintech. Proporciona una solución para la administración de activos financieros, permitiendo a los usuarios gestionar su saldo, realizar depósitos, transferencias y administrar contactos dentro de una interfaz moderna y responsiva.

## Funcionalidades Principales

- Landing Page: Pantalla de bienvenida con diseño de alto contraste y acceso al sistema.
- Inicio de Sesión: Flujo de validación para el acceso a la plataforma privada.
- Visualización de Saldo: Monitorización dinámica del capital disponible en tiempo real.
- Gestión de Depósitos: Interfaz para el ingreso de activos con actualización inmediata del estado de cuenta.
- Transferencias: Sistema de envío de fondos a contactos registrados con validación de disponibilidad.
- Historial de Movimientos: Registro cronológico de transacciones con filtros por tipo de operación (Ingresos y Egresos).
- Agenda de Contactos: Módulo para la gestión y almacenamiento de destinatarios frecuentes.

## Tecnologías Utilizadas

- HTML5 y CSS3: Estructura semántica y diseño basado en variables de color y gradientes técnicos.
- Bootstrap 5: Implementación de diseño responsivo, componentes de interfaz y ventanas modales.
- Bootstrap Icons: Integración de iconografía técnica para mejorar la navegación.
- JavaScript y jQuery: Desarrollo de la lógica de negocio, gestión del DOM y control de eventos.
- LocalStorage: Mecanismo de persistencia de datos para mantener la integridad del saldo y el historial entre sesiones.

## Estructura del Proyecto

El desarrollo se organizó mediante una metodología de ramas para asegurar la trazabilidad de las funcionalidades:

- main: Versión estable y final del proyecto.
- feature/login: Desarrollo del módulo de acceso y seguridad.
- feature/depositos: Implementación de la lógica de ingresos y actualización de saldos.
- feature/transacciones: Construcción del sistema de envíos y reportes de historial.
